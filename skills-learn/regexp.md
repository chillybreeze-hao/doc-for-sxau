## 正则表达式介绍

> 正则表达式（Regular Expression，简称 regex 或 regexp）是一种用于匹配字符串中特定模式的工具。它由一系列字符和特殊符号组成，可以用来描述复杂的文本模式。正则表达式在文本处理、数据验证和搜索等领域都有广泛应用。许多编程语言都支持正则表达式，例如 Python、JavaScript 和 Java 等。
